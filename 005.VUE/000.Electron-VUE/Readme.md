@(electron-vue, vue-material)

[TOC]

### 1. 搭建环境

```powershell
# 安装vue-cli
npm install -g vue-cli
# 安装模板脚手架
vue init simulatedgreg/electron-vue my-project

# 安装依赖库
cd my-project
npm install

# 安装vue-material
npm install vue-material --save
```
